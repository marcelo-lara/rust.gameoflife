# Game of Life - Rust

## setup

install wasm-pack
```
curl https://rustwasm.github.io/wasm-pack/installer/init.sh -sSf | sh
```

install node dependencies
```
cd www
npm install
```

## build'n run

```
wasm-pack build
cd www
npm run start
```
